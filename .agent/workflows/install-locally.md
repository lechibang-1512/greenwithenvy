# Install EnvyControl Locally

1. From the repository root, install globally via pip:

```bash
sudo pip install .
```

2. Verify installation:

```bash
envycontrol --version
```

3. Now `envycontrol` is available system-wide:

```bash
sudo envycontrol -s <MODE>
```
