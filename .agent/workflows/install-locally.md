# Install GreenWithEnvy Locally

1. From the repository root, install globally via pip:

```bash
sudo pip install .
```

2. Verify installation:

```bash
greenwithenvy --version
```

3. Now `greenwithenvy` is available system-wide:

```bash
sudo greenwithenvy -s <MODE>
```
